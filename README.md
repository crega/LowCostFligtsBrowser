## LowCostFligtsBrowser
This project is using my template for Clean Code using
* .NET Core 5.0,
* Entity Framework 5.0 for backend,
* Angular 11 for frontend,
# This solution containts "boilterplate" files as well
Those files are here to demostrate the full power of clean architeture.
  For further readings about "Clean Architecture" please visit [this link](https://blog.cleancoder.com/uncle-bob/2012/08/13/the-clean-architecture.html).

Make sure you check out Open API 3 documentation!


In order to run it

* Please make sure, you have .net Core 5.0 SDK installed
* as well as Node.js

To run project:
* run "npm install" in "WebUI/ClientApp",
* then in same directory run "npm start"
* and in "WebUI" run "dotnet start".
